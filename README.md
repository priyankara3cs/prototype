## Run react in development mode

### npm install

```sh
npm install
```

or

```sh
npm install --force
```

### Development Server

Start the development server on `http://localhost:8000`:

```sh
npm run dev
```

## Production

Build the application for production:

```bash
# npm
npm run build
```
